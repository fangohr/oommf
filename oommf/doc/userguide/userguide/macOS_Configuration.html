macOS Configuration

The build procedure for macOS is the same as for Unix. The platform name is ``darwin''. If the platform configuration check does not find a C++ compiler, then you will have to install one. One option is the Xcode command line developer tools provided by Apple. You can install these from a Terminal window via the command 
xcode-select --install
You should run this command even if you install the full Xcode IDE. Refer to your system documentation for details.
